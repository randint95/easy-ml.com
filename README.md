# CSV回帰分析・機械学習ツール

## 概要
ブラウザ上でCSVファイルを読み込み、サーバーレス環境で機械学習アルゴリズムを実行できるWebアプリケーションです。
「手元のデータをサクッと分析したい」「機械学習のアルゴリズムがどのように結果を返すのか学びたい」という目的で作成しました。

## 主な機能
* **データの簡易プレビュー機能**: サーバーにデータを送る前に、ブラウザ上でCSVの中身と文字コード（UTF-8, Shift-JISなど）の確認が可能。
* **重回帰分析**: 複数の説明変数から目的変数（数値）を予測し、各変数の係数（重み）を算出。
* **ロジスティック回帰分析**: 2値分類（例：Yes/No、良品/不良品など）の確率を予測し、正解率を算出。
* **ランダムフォレスト**: 分類・回帰の両方に対応し、各変数の「特徴量重要度」を算出して棒グラフで視覚的に表示。
* **柔軟なデータ前処理オプション**: 
    * ヘッダー（列名）やインデックス（行名）の有無を指定可能。
    * 説明変数のスケールを揃える「データの標準化（StandardScaler）」のON/OFF切り替え機能を実装。

## 使用技術
### フロントエンド
* HTML5 / CSS3
* JavaScript (Vanilla JS)
    * ※外部ライブラリに依存せず、標準の `fetch` APIや非同期処理（`async/await`）を用いて実装。

### バックエンド/インフラ
* Python 3.12
* pandas / scikit-learn / numpy
* AWS (Lambda, API Gateway)

## アーキテクチャと処理の流れ
1.  ユーザーがフロントエンド（ブラウザ）でCSVファイルと各種オプションを選択。
2.  JavaScriptの `FileReader` APIを用いてCSVをテキストデータとして読み込み、JSON形式でAPI Gatewayへ送信。
3.  AWS Lambda (Python) が起動し、`pandas` でデータフレーム化および前処理（ダミー変数化、欠損値除外など）を実行。
4.  `scikit-learn` を用いてモデルの学習・評価を行い、結果（精度、係数、重要度など）をフロントエンドへ返却。
5.  JavaScriptがレスポンスを受け取り、DOM操作によって画面に結果を描画。

## 工夫した点・アピールポイント
* **UX（ユーザー体験）の向上**: 
    機械学習の処理中はユーザーを待たせてしまうため、非同期処理を活用して「処理中...」のローディング画面を実装しました。また、X（説明変数）とY（目的変数）の選択時に、同じ列を重複して選べないようにJavaScript側で排他制御のロジックを組んでいます。
* **堅牢なエラーハンドリング**: 
    「目的変数が数値ではない」「データ行数が少なすぎる」「分類タスクなのに変数の種類が多すぎる」といったユーザー側の操作ミスに対して、バックエンド側で細かく例外処理（`try-except`）を行い、フロントエンドに分かりやすいエラーメッセージを返すように工夫しました。
